# SmartAcad
SmartAcad is a student productivity and campus management platform that integrates notes sharing, assignment reminders, canteen queue management, and library tracking into one system, helping students stay organized and improving overall campus efficiency.
# SmartAcad 🚀

SmartAcad is a student productivity and campus management platform.

## Features
- 📚 Notes Repository
- ⏰ Assignment Reminder
- 🍔 Canteen Queue System
- 📖 Library Tracking
- 🎯 Productivity Reward System

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## Setup

### Backend
cd server
npm install
node server.js

### Frontend
cd client
npm install
npm start
